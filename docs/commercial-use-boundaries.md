# Commercial Use Boundaries

The official Compendium site states that the Adult, Older Adult, and Wheelchair Compendia are free to use for commercial purposes.

This repository still treats redistribution as a separate boundary:

- using or citing the Compendium is allowed under the official source posture;
- importing from a user-provided upstream download is supported;
- redistributing upstream tables from this public GitHub repo requires explicit review before any data is committed.
